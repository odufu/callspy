# CallSpy

**Work in Progress - Updated Daily**

## Project Overview
CallSpy is an Android call recording app that automatically records calls, including WhatsApp and normal calls, and saves them stealthily.

## Architecture
The app is built using a feature-first clean architecture approach with Kotlin.

## Current Status
Currently in the research phase, focusing on understanding Android call recording APIs and restrictions.

## Planned Features
- Automatic call detection
- Call recording
- Stealth storage of recordings
- Playback functionality

## Setup Instructions
1. Clone the repository
2. Open the project in Android Studio
3. Configure necessary permissions for call recording
4. Build and run the app on a compatible Android device
